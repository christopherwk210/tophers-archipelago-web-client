My goal with this project is to provide a nice way to chat with other players in an Archipelago instance from any web capable device.

Features:

- Mobile support
- Basic markdown support for bolding and italics for user chat messages
- Multi-account switching and management
- UI for buying hints with item drop downs
- Themes & custom colors
- Audio notifications
- Sortable & searchable tables
- No download required
- TONS more

Libraries:

- [archipelago.js](https://github.com/ThePhar/archipelago.js)
- [98.css](https://github.com/jdan/98.css)
- [vue](https://github.com/vuejs/core)
