# Topher's Archipelago Web Client

My goal with this project is to provide a simple way to chat with other players in an Archipelago instance from any web capable device.

Features:

- Mobile support
- Messages are parsed & color coded
- Saves previously entered information for future visits
- Can automatically resume connections
- Supports basic markdown bolding and italics
- Automatically converts URLs to clickable links

Libraries:

- [archipelago.js](https://github.com/ThePhar/archipelago.js)
- [98.css](https://github.com/jdan/98.css)
- [vue](https://github.com/vuejs/core)
